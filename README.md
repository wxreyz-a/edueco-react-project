# EduEco

Projet React pour EduEco avec un backend Express (feedback).

## Demarrage

Frontend:

```bash
npm install
npm start
```

Backend:

```bash
cd backend
npm install
npm start
```

## Tests

Frontend:

```bash
npm test
```

Backend:

```bash
cd backend
npm test
```

## Build

```bash
npm run build
```

La commande de build genere automatiquement `public/sitemap.xml` avant la compilation.
